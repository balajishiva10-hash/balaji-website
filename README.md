# Balaji – YouTube UI Clone 

A pixel-perfect YouTube homepage UI clone built using **pure HTML and CSS**.  
This project replicates the layout, spacing, typography, and responsive behavior of YouTube’s homepage — without using JavaScript or any frameworks.



---

##  Features

- Fully responsive YouTube homepage layout  
- Header, sidebar, and video grid recreated  
- Real YouTube-style fonts, spacing, and UI  
- Built using **pure HTML & CSS only**  
- Deployed using **GitHub Pages**

---

##  Tech Stack

- HTML5  
- CSS3  
- Google Fonts (Roboto)  
- GitHub Pages (hosting)

---

##  Purpose of this project

This project was built to:
- Practice frontend UI replication
- Improve CSS layout, positioning, and responsiveness
- Showcase real-world frontend skills on GitHub & LinkedIn
